**This is a sample project that demonstrates the use of post in Node / express to upload images to a server, do not use in production**

Setup Steps:
* Run `docker compose build` to create containers
* You will need to run `/pb/pocketbase superuser upsert youremail@domain.com super-strong-password` after the initial build to create a pocketbase superuser
* Run `npm install` in the /app directory
* Run `npm run build` in the /app directory
